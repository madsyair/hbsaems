# Compare Fitted HBMs

Primary model-comparison functions in hbsaems. `model_compare` handles
one or two models (supersedes deprecated
[`hbmc`](https://madsyair.github.io/hbsaems/reference/deprecated.md));
`model_compare_all` handles N models (analogous to
[`loo_compare`](https://mc-stan.org/loo/reference/loo_compare.html)).
